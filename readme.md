### Primeira utilização

1) Faça o download do framework pelo site oficial; 

	https://www.safepentest.com.br/donwloads/systems/cqm.tar.gz

2) Faça a instalação fornecendo os dados para ativar sua licença;
	
	sh cqm.sh installation
	
3) Inicialize a aplicação:

	sh cqm.sh start
	
4) Crie seus scripts e armazene-os nos locais apropriados;


5) Mantenha o CQM sempre atualizado.
